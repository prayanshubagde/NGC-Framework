## 1. Purpose

This module serves as the **physics layer** for the entire AGI architecture. It does not perform high-level computation itself; instead, it provides the fundamental, immutable laws, constants, and geometric primitives that govern all other modules. It is the system's source of ground truth for geometric reality.

## 2. Dependencies

- **None.** This is the foundational module.

## 3. Core Data Structures

This module primarily exposes constants and validation functions. Its core "data structure" is the definition of the computational space itself.

-   **Coordinate System:** A triaxial polar coordinate system defined by three intersecting planes offset by 120°.
-   **Fundamental Constants:**
    -   `phi = 1.61803398875...` (The Golden Ratio)
    -   `sqrt2 = 1.41421356237...` (The square root of 2)
-   **Geometric Primitive:** The **Tetrahedron**, defined as a set of four 3D vectors forming a closed volume.

## 4. API Specification

### `get_coordinate_system()`

-   **Description:** Returns the parameters defining the system's base coordinate space.
-   **Inputs:** None.
-   **Outputs:**
    -   `Object`: `{ planes: 3, offset_angle: 120, type: 'triaxial_polar' }`
-   **Error Handling:** None.

### `validate_tetrahedron(vectors)`

-   **Description:** Checks if a set of four 3D vectors forms a valid, closed tetrahedral volume according to the rules of projection geometry.
-   **Inputs:**
    -   `vectors` (Array of 4 Arrays): `[[x1,y1,z1], [x2,y2,z2], [x3,y3,z3], [x4,y4,z4]]`
-   **Outputs:**
    -   `Boolean`: `true` if the vectors form a valid tetrahedron, `false` otherwise.
-   **Error Handling:**
    -   Throws `InvalidInputError` if the input is not an array of four 3D vectors.

### `get_phi_scale(level)`

-   **Description:** Returns the projection radius `R_n` for a given recursive shell `n`, based on the formula `R_n = φⁿ`.
-   **Inputs:**
    -   `level` (Integer): The recursive depth or shell number (n ≥ 0).
-   **Outputs:**
    -   `Float`: The calculated projection radius.
-   **Error Handling:**
    -   Throws `InvalidInputError` if `level` is not a non-negative integer.

### `get_master_equation_components()`

-   **Description:** Returns the components of the NGC Master Equation, `π² = (7φ² + √2) / 2`, linking the geometry to fundamental physics.
-   **Inputs:** None.
-   **Outputs:**
    -   `Object`: `{ pi_squared: 9.8696..., seven_phi_squared: 18.35..., sqrt2: 1.414... }`
-   **Error Handling:** None.

## 5. Performance Benchmarks

-   As this module only performs basic calculations and validations, all functions are expected to execute in **O(1)** time.
-   Execution time should be less than 1 microsecond on standard hardware.
